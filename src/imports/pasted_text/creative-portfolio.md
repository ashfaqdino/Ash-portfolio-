Design and generate a premium, modern, visually polished portfolio website for a creative digital agency/company. The purpose of this website is to professionally showcase all design and creative work in a highly organized and user-friendly way. The website should present different categories of work such as websites, mobile applications, social media posts, branding, videos, and other creative assets in a clean and elegant gallery-based structure.

The website must feel high-end, minimal, modern, and visually immersive. The design should follow strong UI/UX principles, with clean spacing, strong hierarchy, excellent readability, smooth transitions, and a highly intuitive browsing experience. The goal is to make the company look professional, creative, and design-focused.

Core Website Goal

Create a portfolio website where users can easily browse creative work by category. Each category should feel like a dedicated portfolio collection. Users should be able to:

See all major design categories on the main portfolio screen
Click on any category to open that section
View image and video previews related to that category
Click any preview to open the selected item in fullscreen
Close the fullscreen view and return to the same gallery screen without losing position
Website Structure

The website should include the following main sections:

Home / Landing Section
Portfolio Categories Section
Individual Category Gallery Pages
Fullscreen Preview Experience
About / Company Intro Section
Contact / Inquiry Section
Main Categories to Include

Create dedicated visual sections for:

Website Design
Mobile Application Design
Social Media Posts
Branding / Identity Design
Video / Motion Design

Each category should be clearly separated and visually represented with an elegant card or tile layout. On hover, each category card should feel interactive and premium with subtle motion, glow, scale, or image animation.

User Flow

The expected user flow should be:

User lands on homepage
User sees a premium hero section introducing the company
Below that, user sees portfolio categories in a clean visual layout
When user clicks a category, the website opens that category’s gallery
Inside the gallery, previews of images and videos are shown in a stylish responsive grid
If user clicks on any preview, that item opens in a fullscreen modal/lightbox
The fullscreen preview must include:
prominent close/cross button at the top right
dark or blurred background overlay
easy next/previous navigation if possible
image shown in large size or video playable within fullscreen
When the user clicks the cross button, the fullscreen closes and the user returns to the same gallery page and same browsing state

This interaction must feel smooth, premium, and professional.

Design Style / Theme

Use a modern luxury creative agency aesthetic. The theme should feel sleek, elegant, minimal, and visually strong.

Recommended visual style:

Dark premium background or a balanced modern dark-light combination
Background tones like deep black, charcoal, graphite, or soft off-white with elegant contrast
Accent color can be electric blue, royal blue, deep red, purple, or gold depending on premium feel
Typography should be modern, clean, and bold for headings, with refined sans-serif body text
Large visual cards
Soft rounded corners
Smooth shadows or subtle glassmorphism if suitable
Plenty of whitespace
Smooth micro-interactions and hover animations
Minimal clutter, focus should remain on creative work

The overall look should feel like a mix of a premium design studio, modern agency, and elegant showcase platform.

Homepage Requirements

The homepage should have:

A strong hero section with bold heading, short company introduction, and a premium visual feel
A clear CTA like “View Our Work” or “Explore Portfolio”
A featured categories section with interactive cards
Optional featured projects preview
Elegant section transitions
A concise about section introducing the company and its design expertise
A contact section for inquiries or project discussion
Portfolio Categories Section

The category section is the most important part of the website. It should:

Display all portfolio categories in a neat and premium layout
Use large cards with category title, cover image or abstract preview
Show subtle hover interaction
Clearly indicate that the card is clickable
Open the corresponding category page/gallery on click
Gallery Page Requirements

Each category gallery page should be optimized for showcasing visuals. It should:

Have a clean page title with breadcrumb or back navigation
Show a grid or masonry layout of previews
Support both image and video thumbnails
Maintain visual consistency across all previews
Allow clicking on any card/thumbnail to open fullscreen preview
Load beautifully on desktop, tablet, and mobile

The gallery should feel smooth, spacious, and immersive. The focus should remain on visual presentation.

Fullscreen Preview / Lightbox Behavior

This is a critical feature and must be designed properly.

When a user clicks any preview:

Open the selected image or video in fullscreen modal/lightbox
Background should dim or blur
Content should appear centered and large
There must be a visible close/cross button at the top right
Closing the modal should return the user to the same gallery page, not homepage
Modal transitions should be smooth and elegant
Videos should be playable in modal
Images should fit nicely without distortion
Optional arrows for next/previous navigation are preferred
Touch-friendly interactions should exist on mobile

The fullscreen experience must feel polished, cinematic, and user-friendly.

The design should strongly follow UI/UX best practices:

Simple and intuitive navigation
Clear visual hierarchy
Consistent spacing and alignment
Fast content discoverability
Responsive design for all devices
Modern hover states and transitions
Easy back and close actions
Accessibility-conscious sizing and contrast
Clean typography and readability
No cluttered layout
Strong visual focus on the company’s work
Animation and Interaction Style

Use subtle but high-quality animations:

Smooth page transitions
Hover zoom or lift effects on cards
Soft fade-in animations on sections
Elegant modal opening animation
Smooth scrolling
Refined button interactions
Premium feel without overdoing motion

Animations should make the website feel alive but still professional.

Content Presentation

The website should be built to visually showcase the company’s design work. Every preview card should be presentation-first. The interface should make images and videos look valuable and premium.

If project details are shown, keep them minimal and elegant:

Project title
Category
Short one-line label if needed

The main focus should stay on visuals, not heavy text.

Desktop should feel cinematic and spacious
Tablet should preserve layout elegance
Mobile should remain easy to browse with swipe-friendly previews and fullscreen viewing
